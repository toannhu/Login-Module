#HOMETEST

Signup and login module using safely generate and verify bcrypt password hashes.

## Prerequisites

- Java SDK 1.7 or above
- Maven

## Build

```
mvn install
```

## Running

```
java -jar ./dist/HomeTest-1.0-jar-with-dependencies.jar
```
